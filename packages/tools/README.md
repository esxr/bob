# Tools Package

Sample MCP tools for the Bob Agent.

## Available Tools

### 1. Calculator
Basic arithmetic operations

### 2. Web Search
Simple web search capability

### 3. File Operations
Read/write file operations

## MCP Server Configuration

The tools are exposed via MCP protocol for standardized access by the agent.

## Setup

```bash
npm install
```

## Usage

The tools are automatically available to the agent via MCP integration.
